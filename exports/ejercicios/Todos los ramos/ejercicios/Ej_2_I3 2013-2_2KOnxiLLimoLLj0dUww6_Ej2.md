Calcular el trabajo realizado por la fuerza  − →  F   ( x, y ) = ( x   −   y x 2   +   y 2   ) − →  i   + (   x x 2   +   y 2   −   y ) − →  j  al ir desde el punto   (1 ,   0)   al punto   (1 + 2 π,   0)   a lo largo de la curva  − → r   ( t ) = (1 +   t   cos( t ) , t   sin( t )) .  Solucion:   Pongamos   P   = ( x   −   y x 2 + y 2   )   y   Q   = (   x x 2 + y 2   −   y ) .   Se comprueba que   ∂Q ∂x   =   ∂P ∂y   en  R 2   \   (0 ,   0) . Considere los arcos siguientes orientados segun su parametrizacion:  C   :   − → r   ( t ) = (1 +   t   cos( t ) , t   sin( t ))   0   ≤   t   ≤   2 π. C 1   :   − → r   1 ( t ) = ( t,   0)   1   ≤   t   ≤   1 + 2 π. C 2   :   − → r   2 ( t ) =  1 2 (cos( t ) ,   sin( t ))   0   ≤   t   ≤   2 π.  Aplicando el Teorema de Green a la region encerrada por la union de estos arcos y tomando en cuenta el sentido en que son recorridos se tiene  ∫  C  =  ∫  C 1  +  ∫  C 2  La dos ultimas integrales se calculan directaments obteniendose  ∫  C 1  =  1 2 ((1 + 2 π ) 2   −   1) = 2 π (1 +   π ) ,  ∫  C 2  = 2 π.  Consecuentemente,   ∫  C   = 2 π (1 +   π ) + 2 π   = 2 π (2 +   π ) .
