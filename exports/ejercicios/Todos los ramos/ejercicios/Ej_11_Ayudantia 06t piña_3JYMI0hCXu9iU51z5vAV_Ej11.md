Demuestre que si  ∞ ∫  a  | f   ( x ) | d x   es convergente, entonces  ∞ ∫  a  f   ( x ) d x   converge (donde   a   es positivo). Solución: Sabemos que:  −| f   ( x ) | ≤   f   ( x )   ≤ | f   ( x ) |  Empleando el criterio de comparación de integrales, y ya que   f   ( x )   ≤ | f   ( x ) |   se tiene que si  ∞ ∫  a  | f   ( x ) | d x   converge, entonces  ∞ ∫  a  f   ( x ) d x   converge.  Observaciones:  Si la integral de   | f   ( x ) |   converge se dice que la integral de   f   ( x )   converge absoluta- mente . Si la integral de   f   ( x )   converge y la integral de   | f   ( x ) |   diverge, se dice que   f   ( x )   converge condicionalmente . 24
