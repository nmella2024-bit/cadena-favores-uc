Determine si la siguiente afirmaci´ on es verdadera o falsa; de ser verdadera, demu´ estrela; y si no d´ e un contraejemplo que la refute: Si   f   y   g   son continuas en [ a, b ] entonces  ∫   b a  ( f   ( x )   g ( x ))   dx   =  (∫   b a  f   ( x )   dx  ) (∫   b a  g ( x )   dx  )  .  Soluci´ on:  La afirmaci´ on es falsa, casi cualquier par de funciones continuas que uno elija permite verificar esto.  Ejemplo:   Sean   f   ( x ) =   x ,   g ( x ) =   x 2 ,   a   = 0,   b   = 1. Entonces, por una parte,  ∫   b a  ( f   ( x )   g ( x ))   dx   =  ∫   1 0  ( x   ·   x 2 )   dx   =  ∫   1 0  x 3   dx   =  1 4 y por otra  (∫   b a  f   ( x )   dx  ) (∫   b a  g ( x )   dx  )  =  (∫   1 0  x dx  ) (∫   1 0  x 2   dx  )  =  1 2   ·   1 3  =  1 6 .
