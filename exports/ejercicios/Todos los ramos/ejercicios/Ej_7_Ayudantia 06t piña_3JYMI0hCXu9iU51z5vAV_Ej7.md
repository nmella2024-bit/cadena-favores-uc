Solución: Se tiene que   ∞ ∑  n =1  a n   converge, por esta razón se desprende que por la prueba de convergencia de series:  l´ ım  n →∞   a n   = 0  Ahora, apliquemos el criterio de comparación al límite. Para ello usamos una serie cono- cida: utilizaremos la serie de   a n   que sabemos que sí converge. Para el caso de   ∞ ∑  n =1  a n  a n +1   se tiene que:  L   = l´ ım  n →∞  a n  1+ a n  a n  = l´ ım  n →∞  1 1 +   a n  =   1 1 + l´ ım  n →∞   a n  =   1 1 + 0  = 1  De esta forma, ya que   L   = 1   y   0   <   1   <   ∞   se tiene que la serie   ∞ ∑  n =1  a n  a n +1   converge. Para el caso de   ∞ ∑  n =1  a n  1 − a n   se tiene que:  L   = l´ ım  n →∞  a n  1 − a n  a n  = l´ ım  n →∞  1 1   −   a n  =   1 1   −   l´ ım  n →∞   a n  =   1 1   −   0  = 1  De esta forma, ya que   L   = 1   y   0   <   1   <   ∞   se tiene que la serie   ∞ ∑  n =1  a n  1 − a n   converge. 17
