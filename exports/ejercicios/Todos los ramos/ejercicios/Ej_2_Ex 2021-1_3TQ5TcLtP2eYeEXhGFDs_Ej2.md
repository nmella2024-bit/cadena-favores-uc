Sean   a >   0 ,   b >   0   y   f   la función definida por:  f   ( x ) =  ∫   b a  t x dt.  Calcule   f   ( − 1)   y demuestre que   f   es continua en   x   =   − 1 .  Solución:  Observe que de la definición tenemos que  f   ( − 1) =  ∫   b a  t − 1 dt   = ln( x ) | b a   = ln( b )   −   ln( a ) = ln  (   b a  )  .  Por otra parte tenemos que si   x ̸   =   − 1 ,  ∫   b a  t x dt   =  (   t x +1  x   + 1  )  | b a   =   b x +1   −   a x +1  x   + 1  por lo que  lim  x →− 1   f   ( x ) =   lim  x →− 1  b x +1   −   a x +1  x   + 1   .  Observamos que el último de estos límites es de la forma indeterminada   0 / 0   por lo que podemos aplicar L’Hôpital obteniendo que  lim  x →− 1   f   ( x )   =   lim  x →− 1  b x +1   −   a x +1  x   + 1 =   lim  x →− 1  ln( b ) b x +1   −   ln( a ) a x +1  1 =   ln( b )   −   ln( a ) =   ln  (   b a  )  .  Por lo tanto, como   lim  x →− 1   f   ( x ) =   f   ( − 1)   tenemos que   f   es continua en   x   =   − 1 .
