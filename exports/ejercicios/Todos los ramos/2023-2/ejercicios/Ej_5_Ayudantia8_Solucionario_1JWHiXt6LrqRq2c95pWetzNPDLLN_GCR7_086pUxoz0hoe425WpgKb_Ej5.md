Demuestre que la ecuación   arctan( x   −   1) +   x 3   −   3 = 0   tiene una única raíz.  Solución  Considere   f   ( x ) = arctan( x   −   1) +   x 3   −   3   y note que   f   es continua en   R ,  f   (0) = arctan( − 1)   −   3 =   − π  4   −   3   <   0   y   f   (2) = arctan(1) + 8   −   3 =   π  4   + 5   >   0 , entonces, por el teorema del valor intermedio, existe un valor tal que   f   ( c ) = 0 , lo cual garantiza la existencia de la raíz. Para la unicidad, suponga que existen dos raíces,   r 1   y   r 2 , entonces,   f   ( r 1 ) =   f   ( r 2 ) = 0   y como  f   es continua y derivable en   R , por el teorema de Rolle, existe un valor   c ,   c   ∈   ( r 1 , r 2 )   tal que  f   ′ ( c ) = 0   pero,   f   ′ ( x ) =   1 1+( x − 1) 2   + 3 x 2   >   0 , entonces   f   ′ ( c ) = 0   implica una contradicción que proviene de suponer que existen dos raíces de la ecuación, por lo que se concluye que la raíz es única.
